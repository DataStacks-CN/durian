# Durian

Durian 是一个 Java 工具类集合：

* Bean：字典（Map）转换或填充对象（JavaBean）工具类
* Constant：常量工具类
* Datetime：日期和时间处理工具类
* Json：Json 处理工具类
* Uuid：UUID 工具类
* MD5：MD5 工具类

## 下载

```xml

<dependency>
    <groupId>cn.datastacks</groupId>
    <artifactId>durian</artifactId>
    <version>0.9.1-SNAPSHOT</version>
</dependency>
```
